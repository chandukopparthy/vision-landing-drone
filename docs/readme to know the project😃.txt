Vision-Landing Drone
About the Project

Vision-Landing Drone is an AI-powered autonomous drone landing system developed using Afferens Physical Perception API. The drone streams live visual data, analyzes the environment in real time to identify a safe landing zone, and automatically sends actuation commands to perform a safe and accurate landing. The project demonstrates how AI can bridge the gap between software intelligence and real-world robotics through physical perception.

How to Run
Download or clone the project.
Install all the required Python libraries listed in the project requirements.
Once the installation is complete, simply double-click the provided .bat file.
The batch file will automatically start all the required services and launch the project.
